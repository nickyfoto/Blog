<h2 id="toc_3">Mortgage</h2>

<p>因为还款额不变，所以每个月要计算利息的金额一直在变。这里求X的值有两种计算方法。一种是：</p>

<p>mx = loan + totalInterests</p>

<p>例如如果只借一个月，那么</p>

<p>mx = loan + loan*R, m = 1</p>

<p>也就是：x = loan*(1+R)</p>

<p>如果借两个月</p>

<p>mx ＝ loan ＋ 一月利息 ＋ 二月利息</p>

<p>mx = loan + loan*R + (loan*(1+R)-x)*R</p>

<p>合并一下</p>

<p>mx = loan*(1+R) + loan*R*(1+R)-R*x = loan*(1+R)<sup>2</sup> -R*x, 当 m ＝ 2 时：</p>

<p>2x = loan*(1+R)<sup>2</sup> - x*R</p>

<p>x = loan*(1+R)<sup>2</sup>/(2+R)</p>

<p>这时，如果我等号右边，上下都乘以R</p>

<p>x = loan*(1+R)<sup>2</sup>/(2+R)＝loan*R*(1+R)<sup>2</sup>/2R+R<sup>2</sup>=loan*R*(1+R)<sup>2</sup>/(R+1)<sup>2</sup>-1</p>

<p>如果你继续算下去，以此类推：</p>

<p>如果分三个月还款：</p>

<p>x = loan*R*(1+R)<sup>3</sup>/(R+1)<sup>3</sup>-1</p>

<p>如果分m个月还款</p>

<p>x = loan*R*(1+R)<sup>m</sup>/(R+1)<sup>m</sup>-1</p>

<p>这也就是我们代码的第一个function的来源。</p>

<div><pre><code class="language-python">def findPayment(loan, r, m):
    &quot;&quot;&quot;Assume: loan and r are floats, m an int
    Returns the monthly payment for a mortgage of size
    loan at a monthly rate of r for m months&quot;&quot;&quot;
    
    return loan*((r*(1+r)**m)/((1+r)**m-1))</code></pre></div>

<div><pre><code class="language-python">class Mortgage(object):

<p>在介绍了pylab之后，TA做了一个视频，介绍了python function中的default keywords arguments. 总结一下：</p>

<ol>
<li><p>python function中的parameter，如果没有默认值，在call这个function的时候要全部给出。但又时候很多参数不是每次call的时候都需要改变，这时就可以用赋值的方式把它的默认值写上。</p></li>
<li><p>有了默认值的function在使用的时候有一定规则。non-keyword arguments要先给出，之后再给需要修改的默认值。默认值可以按顺序赋值，也可以用<code>=</code>来直接赋值。</p></li>
<li><p>当遇到参数比较多的function时，有时在call这个function时可以把non-keyword参数也用<code>=</code>来赋值，这样比较容易读懂给定参数会被用在哪里。</p></li>
</ol>

<p>这里顺便提一下在函数定义中经常看到的<code>*args</code>和<code>**kwargs</code>。<code>*</code>表示多。前者表示参数的长度不确定，后者表示有多个keyword arguments会给入。</p>

<p>拿Python官方文档做个<a href="https://docs.python.org/2/tutorial/controlflow.html#arbitrary-argument-lists">例子</a>：</p>

<div><pre><code class="language-python">def cheeseshop(kind, *arguments, **keywords):

    print &quot;-- I&#39;m sorry, we&#39;re all out of&quot;, kind
    for arg in arguments:
        print arg
    print &quot;-&quot; * 40
